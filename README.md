# 🌍 Geomagnetic Observatory Data Analysis

A Python-based project for analyzing geomagnetic observatory data to study variations in Earth’s magnetic field, including geomagnetic disturbances, secular variation, and storm-time responses.

This project focuses on transforming raw observatory datasets into meaningful scientific insights through preprocessing, visualization, and signal analysis.

---

## 🚀 Features

- 📊 Load geomagnetic observatory datasets (e.g., IAGA-2002, WDC formats)
- 🧹 Data cleaning and preprocessing (missing values, filtering, alignment)
- 📈 Time-series visualization of magnetic field components (X, Y, Z, F)
- 🌪️ Detection of geomagnetic disturbances and storm events
- 📉 Trend analysis for secular variation studies

---

## 🧪 Data Sources

This project is designed to work with commonly used geomagnetic observatory datasets such as:

- INTERMAGNET observatories (definitive & quasi-definitive data)
- World Data Center (WDC) geomagnetic data
- IAGA-2002 format observatory data

---

## 🧰 Tech Stack

- Python 🐍
- NumPy
- Pandas
- Matplotlib / Seaborn
- SciPy

---

## 📊 Example Analysis

The pipeline typically includes:

1. Import raw geomagnetic data
2. Convert timestamps & standardize format
3. Filter noise and remove invalid readings
4. Plot magnetic field components over time
5. Identify anomalies or geomagnetic storm periods
6. Extract statistical or frequency-domain features

---

## 📈 Goals

- Understand Earth’s magnetic field dynamics from observatory data
- Build reproducible workflows for space weather analysis
- Prepare dataset for advanced modeling (ML / statistical forecasting)
